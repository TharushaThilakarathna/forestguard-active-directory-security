Troubleshooting Tips:

  • Check if the AD server is reachable from this machine
  • Verify that the LDAP service is running on the server
  • Try different username formats:
    • domain\username
    • username@domain.com
    • CN=username,DC=domain,DC=com
  • Make sure the Base DN is correct (use Test Connection to see available DNs)
  • Check if the server requires SSL/TLS (use ldaps:// prefix)
  • Verify that the user has permissions to query the directory
